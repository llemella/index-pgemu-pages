# index-pgemu-pages

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容不针对任何特定域名或网站，仅作为静态资源存储与展示的通用空间。

## 目录说明

```
.
├── index.html          # 仓库主页（可选）
├── pages/              # 归档页面存放目录
│   ├── example1.html   # 示例页面1
│   └── example2.html   # 示例页面2
└── README.md           # 本文件
```

- `pages/` 目录下存放所有归档的 HTML 页面文件。
- 每个页面文件独立存在，互不依赖。

## 页面归档说明

- 所有页面均为静态 HTML 文件，不包含后端逻辑。
- 页面资源（如 CSS、JavaScript、图片等）建议内联或使用相对路径引用，以确保在 GitHub Pages 环境下正常访问。
- 归档页面可能随时间更新或移除，恕不另行通知。

## 维护说明

- 本仓库由维护者不定期更新，用于存放临时或长期需要的 HTML 页面。
- 如需提交新的归档页面，请通过 Pull Request 提交，并确保页面内容符合开源社区规范。
- 仓库不保证所有页面永久可用，也不承担因页面内容引发的任何责任。

## 使用方式

你可以通过以下方式访问已归档的页面：

- 直接克隆本仓库到本地，用浏览器打开对应 HTML 文件。
- 若仓库已启用 GitHub Pages，可通过 `https://<用户名>.github.io/index-pgemu-pages/pages/<文件名>.html` 访问（需自行替换 `<用户名>` 和 `<文件名>`）。

## 许可

本仓库内容按原样提供，不附带任何明示或暗示的担保。具体页面可能包含各自的许可声明，请以页面内标注为准。
